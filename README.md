## 🚀 Featured Projects

### [Premier League Analytics Platform](https://github.com/aqshayc-hash/premier-league-analytics)
> **Dagster · dbt · PostgreSQL · Streamlit**

End-to-end data engineering project built with the modern data stack. Ingests match data, xG metrics, and player stats from multiple APIs, transforms through dbt (staging → intermediate → marts), and surfaces in a 5-tab Streamlit dashboard covering standings, form, title race, relegation battles, and player performance.

- Fully containerised with Docker Compose
- Scheduled ingestion + pipeline via Dagster
- dbt tests, sources, and documentation included
- Pytest with mocked API clients

### [Gmail Interview Tracker](https://github.com/aqshayc-hash/Gmail-interview-tracker)
> **React · TypeScript · Tailwind CSS · Gmail API**

A job application tracker that syncs with Gmail via OAuth to automatically detect and track applications. Built for my own job search — four views: Table, Kanban (drag & drop), Calendar, and Stats with funnel charts and activity timelines.

- GIS OAuth, `gmail.readonly` scope — no data stored server-side
- State managed with `useReducer` + localStorage persistence
- Built with Vite 2 for Node 12 compatibility
